# cc15 BreakQuest mobile app
A mobile app that helps counter gaming addiction by temporarily locking games and redirecting users to complete real-world tasks before access is restored. The goal is that over time, users will naturally forget to return to the game.

---

## How It Works

1. User opens BreakQuest instead of their game
2. They pick a real-world quest (walk, clean room, drink water, etc.)
3. A timer runs while they complete the task
4. After hitting the daily minimum productivity duration, the game is "unlocked"

---

## Features

- Game lock with a configurable unlock goal
- Quest selection with multiple task types and durations
- Countdown timer with pause/resume
- Settings to add/delete quests and set daily goals

---

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) 
- [Expo Go](https://expo.dev/client) app installed on your phone

### Installation

```bash
# Clone the repo
git clone https://github.com/jiaying013/cc15.git

# Go into the project folder
cd cc15

# Install dependencies
npm install

# Start the app
npx expo start
```

Scan the QR code in your terminal with the **Expo Go** app on your phone.
